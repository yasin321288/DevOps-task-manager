# DevOps Task Manager

## Overview
DevOps Task Manager is a full-stack application designed to demonstrate real-world DevOps practices by automating the build, deployment, and hosting process using modern tools.

This project focuses on implementing a complete CI/CD pipeline and deploying a containerized application on the cloud.

---

## Objective
To showcase how a development workflow can be automated using DevOps tools, ensuring faster, reliable, and scalable deployments.

---

## Tech Stack

- **Frontend & Backend:** Full-stack web application
- **Containerization:** Docker  
- **CI/CD Pipeline:** Jenkins  
- **Cloud Platform:** AWS (EC2)  
- **Version Control:** Git & GitHub  

---

## Key Features

- Containerized application using Docker  
- Automated CI/CD pipeline with Jenkins  
- Deployed on AWS EC2 instance  
- Automated build and deployment process  
- Scalable and production-like setup  

---

## Workflow

1. Developer pushes code to GitHub  
2. Jenkins pipeline is triggered automatically  
3. Application is built and containerized using Docker  
4. Docker container is deployed on AWS EC2  
5. Application runs live on the cloud  

---

## What This Project Demonstrates

- Continuous Integration & Continuous Deployment (CI/CD)  
- Infrastructure automation  
- Container-based deployment  
- Real-world DevOps workflow  
- Cloud deployment practices  

---

## Quick Summary

This project demonstrates how to automate the deployment of a full-stack application using Docker, Jenkins, and AWS — ensuring faster delivery, scalability, and reliability.
